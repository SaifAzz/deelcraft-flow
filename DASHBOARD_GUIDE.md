# Mind-Links Client Dashboard - User Guide

## Overview

The Mind-Links Client Dashboard is a comprehensive platform for managing contractors, contracts, payroll, payments, and compliance. This guide will help you navigate and use all features effectively.

## Quick Start

### 1. Accessing the Dashboard
- Navigate to `/client-dashboard` in your browser
- You'll see the main dashboard with key metrics and quick actions

### 2. Getting Help
- Click the **Help Circle icon** (?) in the header to access:
  - Quick Start Guide
  - Features Guide
  - Tips & Best Practices

## Main Features

### Dashboard
The dashboard provides an overview of your business:
- **Active Contracts**: Number of ongoing contracts
- **Total Contractors**: All contractors in your system
- **Escrow Balance**: Available funds for payments
- **Pending Approvals**: Items requiring your attention

**Quick Actions:**
- Invite Contractor
- Create Contract
- Create Payroll Run

**Recent Activity**: See latest updates and events
**Upcoming Deadlines**: Important dates and reminders
**Recent Transactions**: Latest payment activity

### Contractors Management

**Inviting a Contractor:**
1. Click "Invite Contractor" button
2. Enter contractor email, name, and optional role
3. Click "Send Invitation"

**Managing Contractors:**
- View all contractors in a table
- Filter by status (Active, Pending, Incomplete)
- Filter by country
- View compliance status
- See active contracts and total earned per contractor

### Contracts

**Creating a Contract:**
1. Click "Create Contract"
2. Follow the wizard:
   - **Step 1**: Select Contractor
   - **Step 2**: Choose Type (Fixed or Hourly)
   - **Step 3**: Set Terms (Amount/Rate, Currency, Dates)
   - **Step 4**: Create Milestones (for Fixed) or Set Rate (for Hourly)
   - **Step 5**: Review & Sign using Deel template

**Contract Types:**
- **Fixed Contract**: Set total amount with milestones
- **Hourly Contract**: Pay based on hours worked at set rate

**Managing Contracts:**
- View all contracts with status (Active, Draft, Completed)
- Track progress and milestones
- Approve completed milestones

### Payroll Management

**Creating a Payroll Run:**
1. Click "Create Payroll Run"
2. Select period (month/year)
3. System generates line items automatically
4. Review and add adjustments if needed
5. Approve payroll
6. Process payments

**Payroll Features:**
- View upcoming payroll
- See total amount and contractor count
- Add adjustments (bonuses, deductions, corrections)
- Track processed payroll history

### Payments & Escrow

**Managing Escrow:**
- View current escrow balance
- Add funds to escrow account
- Monitor transaction history

**Transaction Types:**
- **Payments**: Outgoing payments to contractors
- **Funding**: Money added to escrow

### Compliance Monitoring

**Tracking Compliance:**
- View compliance overview (Fully Compliant, Incomplete, Expiring)
- Monitor contractor compliance status
- Track KYC status
- See missing documents
- View compliance scores

**Compliance Dashboard:**
- See all contractors' compliance status
- Filter by compliance state
- Generate compliance reports

## Using Deel Contract Templates

The dashboard includes Deel-style contract templates for professional agreements.

**Contract Template Includes:**
- Standard contractor agreement structure
- Service description section
- Compensation terms
- Term and termination clauses
- Independent contractor provisions
- Confidentiality agreements
- Intellectual property assignments
- Governing law

**Generating a Contract:**
1. Create a new contract
2. Fill in all required fields
3. The system automatically generates the contract using the Deel template
4. Review and sign the contract

## Best Practices

### Contract Management
- Set clear milestones with specific deliverables
- Review and approve milestones promptly
- Use Deel templates for standardized contracts

### Payroll Processing
- Review payroll runs before approval
- Add adjustments for bonuses or corrections
- Process payroll on time

### Compliance
- Monitor compliance status regularly
- Set reminders for expiring documents
- Ensure all contractors have complete documentation

### Escrow Management
- Maintain sufficient escrow balance
- Add funds proactively
- Monitor transaction history
